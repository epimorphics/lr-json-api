LR JSON APIs

PPD

PPD Data Documentation

Get price, date, address.

GET /ppd?town={town}&roadname={roadname}&number={number}
  • {town} – can be empty - defaults to "bristol"
  • {roadname} – required
  • {number} – optional

HPI

HPI Data Documentation

Get the HPI index for the last 12 months

GET /hpi/{region}
  • {region} – Name of region (required) e.g. "city-of-bristol"
